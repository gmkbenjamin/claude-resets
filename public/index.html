Every tracked announcement, newest first. Resets flush the counters; limit changes move the ceiling without flushing anything, so they are listed but never counted as resets.

Where the data comes from. Each entry links to the original post. Every timestamp on this site is derived from the post's own X status ID rather than transcribed, so the dates cannot drift from the source.

The Codex event list is catalogued by codex-resets.com, which is also the inspiration for this page; timestamps were re-derived independently. Claude entries were collected from @ClaudeDevs.

Open source. The site, the dataset and the build script all live at github.com/gmkbenjamin/claude-resets — spotted a reset that's missing, or one listed wrongly? Open an issue or a pull request.

Not affiliated with Anthropic or OpenAI. Nothing here predicts a future reset — resets are discretionary, and the only limits you can count on are the ones /usage shows you.
